Core Infrastructure Matrix

⚙️ /setup Configuration

The ultimate initialization protocol. Designed exclusively for Server Administrators to map the backbone of the Phoenix environment. This command writes vital configuration data directly to our MongoDB secure clusters.

  • Interface Deployment Routes: Select your UI environment. Generate a Dedicated Text Channel for clean aesthetics, directly embed panels into the active Voice Channel, or utilize the Hybrid ecosystem for maximum control.
  • Category Allocation Binding: Force all dynamically spawned, zero-latency voice nodes to strictly adhere to a specific root server category, keeping your server hierarchy immaculate.
  • Matchmaking Integration: Designate a centralized text channel. This acts as the global hub where users can algorithmically broadcast their private node invites to gather allies.

🚀 /deploy Protocol

The activation matrix. This command injects our proprietary logic into a standard voice channel, mutating it into a highly intelligent, multi-node Master Hub that listens to state updates instantly.

  • Zero-Latency Dynamic Spawning: The moment a user interacts with the Master Hub, the bot bypasses standard API delays, instantly spawning a temporary, perfectly configured sub-channel and moving the user seamlessly.
  • Custom Architecture (Dynamic): Empowers the user. Deploys the complete UI panel, including the highly requested 'Set Limit' modal, allowing hosts to scale their crowd control dynamically.
  • Fixed Architecture (Strict): For rigid server environments. The Administrator hard-codes a maximum user limit during deployment. The bot's algorithm automatically strips the 'Set Limit' interface from all user panels to ensure absolute compliance.

Operational Control Nodes

✏️ Nomenclature Shift

Launch a secure, encrypted modal interface to dynamically rewrite your node's designation on the fly. To protect Discord's API limits, this function restricts inputs to a 30-character absolute limit and monitors changes via a rigid 10-minute cycle cooldown.

👥 Capacity Scaling

Instantly override the max user threshold of your private node via direct modal input. Need absolute privacy? Lower the limit. Hosting an open lounge? Set the value to '0' to instantly disengage limits and allow infinite scaling across the network.

👢 Tactical Ejection

Maintain absolute authority over your communication hub. Deploy a highly responsive targeting dropdown menu populated with real-time active users. Select any unauthorized or disruptive entity and instantly sever their voice connection.

📝 Telemetry Summary

Execute this command to retrieve an ephemeral, highly detailed intelligence embed. It instantly calculates and displays the Host's identity, the exact population of connected entities, and reads the database timestamp to show precise, down-to-the-second node uptime.

🚫 Absolute Block

Beyond a simple kick, this is a severe security override. Select a target to not only disconnect them instantly but to write their unique Discord ID into the session's blacklist, dynamically rewriting the channel's ACL to explicitly deny access for that specific user.

📂 Restore Clearance

Made a mistake or resolving a conflict? Access the active, real-time registry of blocked individuals through a secure dropdown interface. Selecting a user gracefully wipes their ID from the temporary blacklist and restores their standard connection protocols seamlessly.

✅ Authorize Trust

Elevate up to 5 standard users simultaneously to Trusted VIP status. Trusted entities completely bypass all manual locking mechanisms. The system forces a hardcoded connection overwrite specifically for them, granting entry even during a Perimeter Lockdown.

❌ Revoke Clearance

Security compromised? Open the live trust registry and select a previously whitelisted entity. This action instantly strips their bypass privileges, deletes their specialized permission overwrite, and reverts them to standard, restricted node security rules.

👻 Ghost Protocol

Total stealth mode. Toggling this mechanic manipulates the root visibility permissions for the default server role. Activating it instantly rips your session out of the server grid UI, rendering your operational node completely invisible to unauthorized eyes.

🔒 Perimeter Lockdown

Toggle the absolute physical security seal. When locked, the system explicitly denies connection access to the channel across the server. Only you and your explicitly Authorized Trusted allies can penetrate the node once this rigid lockdown is engaged.

🌸 Global Broadcast

Algorithmically generate a secure, single-use bypass invite link (programmed with a strict Max Age of 300 seconds) and transmit it directly into a beautifully formatted embed within the server's designated Matchmaking Hub. Protected by a stringent anti-spam cooldown.

🎁 Authority Transfer

Need to step away without collapsing the node? Execute a permanent Host transfer. Select any active entity currently sitting in your node to surrender Master Control. The database instantly rewrites the owner tag, passing the entire UI Control Panel safely.

💬 AI Sync Matrix

Awaken the machine. This command toggles the embedded Google Gemini-2.5-Flash AI subsystem. Once engaged, simply mention the bot within your text channel to spawn context-aware, hyper-intelligent, conversational text assistance built directly into your interface.

🗑️ Session Purge

Initiate a total structural wipe. Because this command permanently deletes the node and wipes its data from the MongoDB registry, it is protected by a stringent 15-second bot mention verification system to completely eliminate the risk of accidental deletion by the host.
