Core Infrastructure Design

/setup Configuration

The foundational command utilized by server administrators to establish the initial framework of the Phoenix environment.

  • Interface Deployment: Choose between a Dedicated Channel, Direct Voice Channel embeds, or a Hybrid control panel ecosystem.
  • Category Allocation: Seamlessly bind the deployment parameters to specific server categories to ensure organized hierarchy.
  • Matchmaking Hub Integration: Map an exact text channel where all automated global broadcast invites will be dispatched securely.

/deploy Protocol

The activation matrix that converts a standard voice channel into an intelligent, multi-node Master Hub.

  • Dynamic Node Creation: Automatically spawns temporary, perfectly configured sub-channels the moment a user interacts with the Master Hub.
  • Custom Limit Architecture (Dynamic): Grants absolute freedom to users, providing them with a 'Set Limit' button to scale their active node dynamically.
  • Fixed Limit Architecture (Strict): Enforces a hard-coded user limit defined by the administrator. The 'Set Limit' interface is algorithmically stripped from user panels to ensure strict compliance.

Operational Control Matrix

✏️ Nomenclature Shift

Dynamically modify the designation of your secure active session. Monitored by advanced rate-limiting logic to prevent API exhaust (Restricted to 2 changes per 10-minute cycle).

👥 Capacity Scaling

For systems operating on Custom Architectures, instantly adjust the maximum user threshold of your private node, ensuring optimal crowd control.

👢 Tactical Ejection

Instantly sever the connection of any unauthorized or disruptive entity from your current operational voice node with a single execution command.

📝 Telemetry Summary

Access an ephemeral intelligence readout detailing session host data, current active population, precise node uptime, and current security lock statuses.

🚫 Absolute Block

Not only eject an entity, but write their unique identifier into the session's temporary blacklist, denying any future re-entry attempts.

📂 Restore Clearance

Review the active registry of blocked individuals and gracefully lift restrictions, granting them restored access to the secure node matrix.

✅ Authorize Trust

Elevate standard users to trusted allies. Authorized entities bypass all standard locking mechanisms, allowing seamless entry into secured channels.

❌ Revoke Clearance

Downgrade previously trusted entities back to standard permissions, removing their ability to bypass manual security lockdowns.

👻 Ghost Protocol

Manipulate the server's ViewChannel permissions dynamically. Render your entire session completely invisible to the broader server grid for ultimate privacy.

🔒 Perimeter Lockdown

Toggle the absolute security seal. When locked, the system physically denies connection access to all non-trusted entities attempting to join.

🌸 Global Broadcast

Transmit an elegant, algorithmic invitation beacon directly to the designated matchmaking hub. Protected by a stringent 10-minute cooldown module to prevent spam.

🎁 Authority Transfer

Instantly and securely pass the Master Control hierarchy of your active session to another connected entity via a highly responsive dropdown matrix.

💬 AI Sync Matrix

Activate the embedded Gemini Intelligence sub-system. Once engaged, mentioning the bot within the channel spawns highly sophisticated, context-aware AI interactions.

🗑️ Session Purge

Initiate a total structural wipe. Protected by an interactive 'yes' confirmation collector to prevent accidental erasure of the operational node and its data.
